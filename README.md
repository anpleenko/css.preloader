# css.preloader

[![NPM](https://nodei.co/npm-dl/css.preloader.png?months=1)](https://nodei.co/npm/css.preloader/)

Small CSS preloader

### Usage

1. add preloader.css to your project
1. add class preloader to any HTML block

```html
<div class="some-block preloader"></div>
```
